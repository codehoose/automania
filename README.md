# Automania Retro Remake

This repo contains a Unity version of the [Automania](https://worldofspectrum.org/archive/software/games/automania-mikro-gen-ltd) game from Mikro-Gen.

# Changes

## 2025-11-15 Game Over

- Game over conditions and sorting out what happens when Wally falls from a great height

## 2025-09-26 HUD

- Adding the HUD to let the player know how many lives they have left, what their score is and the all important timer!

## 2025-08-31 Wally Movement

- Wally can move, jump and pick up car parts

## 2025-07-05 Tiled Map Editor

- Added the blocks that will be used to create the levels and blocks for the ZX Spectrum colours. Static blocks will take their colour from these blocks
- Conveyor belt movement - One of my favourite features of this game

## 2025-05-24 ZX Spectrum Colour Clash

- Added a render texture for ink and paper colours that are combined using a shader to give the distinct colour clash that the ZX Spectrum is known for

## 2025-05-07 Initial Upload

- Starting to create the prefabs and the scriptable objects that will be used in the project
